# Jh_Tourism

## Project Overview
Jh_Tourism is a comprehensive tourism management system that aims to provide seamless interaction between travelers and service providers. This project showcases the beauty of Jharkhand by integrating all tourist attractions, accommodation, and activities into a single platform, enhancing the travel experience for users.

## Features
- **User Registration and Login**: Secure user accounts with authentication.
- **Search Functionality**: Users can search for tourist spots, hotels, and activities based on their preferences.
- **Booking System**: Users can book accommodations and activities directly within the platform.
- **Reviews and Ratings**: Users can leave feedback on their experiences and help future travelers.
- **Admin Panel**: An admin interface to manage users, listings, and bookings.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rahulmandal76/Jh_Tourism.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Jh_Tourism
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```  
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- After starting the application, visit `http://localhost:3000` in your browser.
- Create an account or log in to access all features of the application.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Styling**: CSS, Bootstrap
